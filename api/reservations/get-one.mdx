## Endpoint

```
GET /api/reservations/:id
```

## Description

Retrieves a specific reservation by its unique identifier.

See implementation in `server/src/modules/reservations/reservations.controller.ts:145-155`.

## Authentication

<Info>
  This endpoint requires Bearer token authentication. Include your access token in the `Authorization` header.
</Info>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the reservation to retrieve
  
  Example: `res789`
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the reservation
</ResponseField>

<ResponseField name="client_id" type="string">
  ID of the client who made the reservation
</ResponseField>

<ResponseField name="professional_id" type="string">
  ID of the professional for the reservation
</ResponseField>

<ResponseField name="start_time" type="string">
  Start time of the reservation in ISO 8601 format
</ResponseField>

<ResponseField name="end_time" type="string">
  End time of the reservation in ISO 8601 format
</ResponseField>

<ResponseField name="status" type="string">
  Reservation status: `PENDING`, `CONFIRMED`, or `CANCELLED`
</ResponseField>

<ResponseField name="session_modality" type="string">
  Type of session: `Virtual`, `Presencial`, or empty string
</ResponseField>

<ResponseField name="created_at" type="string">
  When the reservation was created (ISO 8601 format)
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET https://your-domain.com/api/reservations/res789 \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json"
```

```typescript TypeScript
const getReservation = async (reservationId: string) => {
  const response = await fetch(`https://your-domain.com/api/reservations/${reservationId}`, {
    method: 'GET',
    headers: {
      'Authorization': `Bearer ${accessToken}`,
      'Content-Type': 'application/json'
    }
  });
  
  const data = await response.json();
  return data;
};

// Usage
const reservation = await getReservation('res789');
```

</CodeGroup>

## Error Responses

### Not Found

If the reservation doesn't exist:

```json
{
  "statusCode": 404,
  "message": "Reserva no encontrada",
  "error": "Not Found"
}
```

### Unauthorized

```json
{
  "statusCode": 401,
  "message": "Token inválido o expirado",
  "error": "Unauthorized"
}
```

## Use Cases

- Display reservation details to users
- Verify reservation information before making changes
- Show confirmation details after booking
- Retrieve reservation data for cancellation or modification
