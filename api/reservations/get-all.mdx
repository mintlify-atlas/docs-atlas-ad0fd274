## Endpoint

```
GET /api/reservations
```

## Description

Retrieves a complete list of all reservations from all users in the system.

See implementation in `server/src/modules/reservations/reservations.controller.ts:70-87`.

## Authentication

<Info>
  This endpoint requires Bearer token authentication. Include your access token in the `Authorization` header.
</Info>

## Request

No parameters required.

## Response

<ResponseField name="status" type="number">
  HTTP status code (200 for success)
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="data" type="array">
  Array of reservation objects
  
  <Expandable title="Reservation object">
    <ResponseField name="id" type="string">
      Unique identifier for the reservation
    </ResponseField>
    
    <ResponseField name="client_id" type="string">
      ID of the client
    </ResponseField>
    
    <ResponseField name="professional_id" type="string">
      ID of the professional
    </ResponseField>
    
    <ResponseField name="start_time" type="string">
      Start time in ISO 8601 format
    </ResponseField>
    
    <ResponseField name="end_time" type="string">
      End time in ISO 8601 format
    </ResponseField>
    
    <ResponseField name="status" type="string">
      Reservation status: `PENDING`, `CONFIRMED`, or `CANCELLED`
    </ResponseField>
    
    <ResponseField name="session_modality" type="string">
      Session type: `Virtual`, `Presencial`, or empty string
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      Creation timestamp in ISO 8601 format
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET https://your-domain.com/api/reservations \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json"
```

```typescript TypeScript
const getAllReservations = async () => {
  const response = await fetch('https://your-domain.com/api/reservations', {
    method: 'GET',
    headers: {
      'Authorization': `Bearer ${accessToken}`,
      'Content-Type': 'application/json'
    }
  });
  
  const data = await response.json();
  return data;
};
```

</CodeGroup>

## Notes

- This endpoint returns all reservations across all users and professionals
- Use [Get By User](/api/reservations/get-by-user) or [Get By Professional](/api/reservations/get-by-professional) for filtered results
- Results may be large depending on system usage
